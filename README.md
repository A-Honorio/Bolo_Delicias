# Bolo_Delicias
Bolo_Delicias é uma plataforma desenvolvida para otimizar a gestão e a venda de bolos, oferecendo uma experiência prática e organizada tanto para a empresa quanto para os clientes. O sistema permite o gerenciamento de pedidos, catálogo de produtos, controle de vendas e acompanhamento do atendimento, tornando o processo mais eficiente e profissional.
