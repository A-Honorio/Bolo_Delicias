# Acompanhamento de Pedidos

**COMO** cliente da Bolo_Delicias,  
**QUERO** acompanhar o status dos meus pedidos,  
**PARA** saber em que etapa minha compra se encontra.  

**Critérios de Aceitação:**
- O sistema deve exibir o histórico de pedidos do cliente.
- Cada pedido deve apresentar status (ex: em preparo, enviado, entregue).
- O sistema deve atualizar o status automaticamente conforme o andamento.
- O cliente deve conseguir visualizar detalhes de cada pedido.
