# Finalização de Pedido

**COMO** cliente da Bolo_Delicias,  
**QUERO** finalizar meu pedido informando endereço e forma de pagamento,  
**PARA** concluir a compra dos produtos selecionados.  

**Critérios de Aceitação:**
- O sistema deve solicitar endereço de entrega.
- O sistema deve permitir escolher forma de pagamento (ex: cartão, PIX, dinheiro).
- O sistema deve validar os dados antes de concluir o pedido.
- O cliente deve receber uma confirmação do pedido após finalização.
