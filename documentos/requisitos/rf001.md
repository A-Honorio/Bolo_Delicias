#Cadastro e Login de Clientes

**COMO** cliente da Bolo_Delicias,  
**QUERO** criar uma conta e realizar login no sistema,  
**PARA** que eu possa fazer pedidos, acompanhar minhas compras e salvar meus dados pessoais.  

**Critérios de Aceitação:**
- O sistema deve permitir cadastro com Nome, E-mail, Telefone e Senha.
- O sistema deve validar se o e-mail já está cadastrado.
- A senha deve possuir requisitos mínimos de segurança.
- O usuário deve conseguir realizar login utilizando e-mail e senha.
- Deve existir funcionalidade de recuperação de senha.
- Após login, o cliente deve ser direcionado para sua área de pedidos.
